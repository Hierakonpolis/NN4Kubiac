# Neural networks tutorials for medical image analysis

![logos](TutorialMUNet/img/logos.png)

These tutorials are designed to introduce you to medical imaging segmentation using convolutional neural network. We do not assume any previous knowledge on CNNs, but at least some familiarity with python is recommended.

[Convolutional Neural Networks](https://github.com/Hierakonpolis/NN4Kubiac/blob/master/TutorialCNN/CNNs.ipynb) [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Hierakonpolis/NN4Kubiac/blob/master/TutorialCNN/CNNs.ipynb)

In the first tutorial we introduce convolutional neural networks, assuming no previous knowledge of the topic. We conclude with a simple implementation of a classifier on the MNIST dataset to build some familiarity with the PyTorch framework.

[Multi-task U-Net](https://github.com/Hierakonpolis/NN4Kubiac/blob/master/TutorialMUNet/MU-Net.ipynb) [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Hierakonpolis/NN4Kubiac/blob/master/TutorialMUNet/MU-Net.ipynb)

Here we go over the entire process of training a convolutional neural network for the skull stripping and region segmentation of the mouse brain. We start from defining dataset wrappers and the data pre-processing for a publicly available dataset, explore the network architecture, layers and loss function, and train the network.
